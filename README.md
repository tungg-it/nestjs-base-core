- Use [NestJS](https://docs.nestjs.com/)
- Use Node v16.14.0
- Use yarn.
- Add variables in .env file based on .env.example file

## Installation

```bash
$ yarn install
```

## Running application

**1. Production.**

```bash
$ start:prod
```

**2. Development.**

```bash
# No watching
$ yarn start
```

```bash
# Watching
$ yarn start:dev
```

**3. Build application.**

```bash
$ yarn build
```

## Lint

```bash
$ yarn lint
```
